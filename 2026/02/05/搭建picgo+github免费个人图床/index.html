搭建picgo+github免费个人图床

图床搭建教程

下载picgo

首先进入github，搜索picgo

如图所示，进入第一个仓库

点击发行版(release)

向下滑动到图片所示位置，根据自己的电脑系统自行选择安装包版本

安装picgo（以Windows系统 64bit 为例）

开启刚才下载的exe文件

选择仅为我安装即可

按照自己的需求调整目标文件夹路径(可选)

新建picgo所需仓库


注意 ：仓库必须为公共（public），以确保可以被访问

配置picgo

打开应用，页面如图所示

点击右侧第五个picgo设置，滑至最底部
选择github即可

选好后，点击右侧第三栏图床设置,按加号区域新建图床

之后按要求填写即可

（1）图床配置名可自定义
（2）设定仓库名与我们刚才新建的仓库名相同
（3）分支写main即可
（4）Token填写放到下一步
（5）储存路径可自行填写，按所给例子填写即可
（6）设置自定义域名
可选用给出的加速域名：https://cdn.jsdelivr.net/gh/
注意：在最后必须加上你自己的仓库名，否则无法加速。
例如，如果你的仓库为 xxx/xxx，则完整的加速域名为：

text

https://cdn.jsdelivr.net/gh/xxx/xxx

重点：获取你的Token

接下来，我们来获取自己的github token
在右上角个人头像处找到设置（Setting）

进入后直接点击左侧栏目最下面的开发者设置（Developer setting）

点开个人访问令牌，经典的即可，点击新建

时间任意，过期后令牌会失效，重新设置即可
注意 :一定要把rupo给打开

以上操作完成后滑到页面最下面，点击创建，把自己的个人令牌复制粘贴下来即可

现在，你的个人免费图床就搭建完成了
注意 ：GitHub的仓库用作图床的存储上限制为1G，用作个人博客应该是足够的，但如果需求量大的话要时常进行更新
